# Inbox

Feature requests and ideas for the todo app.

- [ ] Add a REST API endpoint to create a new todo item (POST /todos with title and optional description)
- [ ] Add a REST API endpoint to list all todo items (GET /todos with optional ?status=done filter)
- [ ] Add a REST API endpoint to mark a todo item as done (PATCH /todos/:id)
- [ ] Add a REST API endpoint to delete a todo item (DELETE /todos/:id)
- [ ] Add input validation — title must be non-empty and under 200 characters
