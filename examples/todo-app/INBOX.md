# Inbox

Feature requests for the todo CLI app. PM triages these into specs and tasks.

- [ ] Add a `todo add "task description"` command that appends a new todo to a JSON file
- [ ] Add a `todo list` command that prints all todos with their index and status
- [ ] Add a `todo done <index>` command that marks a todo as completed
- [ ] Add a `todo remove <index>` command that deletes a todo from the list
- [ ] Add a `todo clear` command that removes all completed todos
- [ ] Add colored output — green for completed todos, white for pending
